Instagram Clone

A front-end Instagram clone made with React.

Instagram is one of today's most popular apps. It features a slick, clean interface along with standout features that make cloning it a great way to show off my React skills.

Working Features

  • Crisp, beautitful UI.
  • Renders fully styled posts with poster info, post image, likes, and comments.
  • Fully working searchbar with search algorithm.
  • Ability to add comments as a logged-in user.
  • Functioning like button.
